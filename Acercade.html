Electro Revolution CADY

Descripción del proyecto:

La aplicación a diseñar tiene como fin el ahorro de energía, cuyo público objetivo a todos los hogares. Dicha aplicación consta de cuatro partes o módulos:

  • 1. El software (aplicación) interactuara con el hardware por medio de señal Bluetooth.
  • 2. La interfaz mostrará los datos almacenados en las baterias obtenidos por medio de la interacción del panel solar.
  • 3. El usuario debe ingresar todos los electrodomésticos de su hogar con los datos de consumo.
  • 4. El software mostrará cómo se distribuyen esos datos o información recogidos del panel solar de manera eficiente y con el fin de ahorrar energía.

Objetivo general:

Crear una aplicación móvil que calcule la energía obtenida mediante paneles solares para ayudar a las personas a utilizarla de manera eficiente, también generar el reconocimiento de la energía alternativa.

Objetivos específicos:

  • Calcular la información sobre la energía recogida.
  • Cumplir con todas las etapas del ciclo del software para lograr un producto de calidad.
  • Ayudar a las personas a economizar energía de una manera ecológica y renovable.
